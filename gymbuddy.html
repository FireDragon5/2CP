Welcome to GymBuddy

GymBuddy is a mobile application that helps you keep track of your workouts and progress. All data is stored online and accessible from anywhere in the world.

Workouts

GymBuddy stores all workouts, displaying them in a list. Users can add new workouts, view details, and delete them.

Personal Best

Track your personal best achievements and view detailed progress on each entry.

Schedule

Gym Buddy has a built in feature that allows the user to create a workout program. The user can also save the program to the database, and access it when needed.
